How to Play

Controls

Use the Arrow Keys (← and → or A and D) to move. Use ↑, W, or Space to jump.

More Help

Gameplay

You control the Player. The goal of this game is to reach the Goal Objects. To do this, you must perform a variety of tasks including:

  • Collecting Keys to unlock Key Blocks
  • Using Plates to change your size, allowing you to reach higher places or fit through smaller gaps;
  • Avoiding Death Blocks

Keys

There are two sets of keys: Green (Green Keys and Green Key Blocks) and Red (Red Keys and Red Key Blocks). When you run into a Key Block, you won't be able to pass through. However, if you had collected a Key, it would open (disappear). You can also have multiple keys at once or a mix of multiple colors of keys. For example, you could collect a Green Key. If you tried to go through a Red Key Block, but it wouldn't work. But if there was a Green Key Block, you could unlock it. But if there is another Green Key Block, you couldn't unlock it because you already used up all of your keys. Then you collect a Green Key and two Red Keys. You could unlock both a Green Key Block and a Red Key Block in any order. But if you try to unlock another Green Key Block, you won't be able to because you already used your Green Key. But if there was a Red Key Block, you could unlock it because you still have one Red Key left.

Plates

There are two types of plates: Small Plates and Large Plates. Plates change the Player's size. Upon death, the Player's size will reset.

Goals

Goals are your main objective to get to. There may be multiple in a level, so sometimes goals will be won't be immediately accessible. Goals can do multiple things, such as:

  • Changing where the Player starts
  • Changing the size that the Player starts at
  • Giving you Keys from the start of the level

Death

Death can be caused by two things: Falling below the level (in some levels) and touching Death Blocks. When you die, you go back to either where you started after you reached you most recent Goal, or where you started the game if you haven't gotten to a Goal yet.

More Controls

  • Press P to record
  • Press Ctrl+O to open a level or extension

Level Editor

To toggle between the level editor and game, press Ctrl+M. The controls are:

  • WASD or ↑←↓→ to move
  • Click an object to select it, or Ctrl click it to select multiple objects
  • Use Ctrl+A to select all objects
  • Press Delete to delete all selected objects
