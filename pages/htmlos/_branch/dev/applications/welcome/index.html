HTML OS, or HyperText Markup Language (the base programming language used on the World Wide Web) Operating System (Windows OS, Mac OS, etc.) is an OS on a web browser.

How do I use HTML OS?

When you start HTML OS, you will see a desktop.

  • By double clicking an icon, you can open a window.
  • By right clicking the desktop, you will see a list of options including:
    • Create file: Creates a file on your desktop with an HTML file, an icon, and a name
    • Reload: Reloads your desktop
    • Settings: Opens the settings menu

Windows

  • Right now, you are reading this in a window.
    • To resize this window, drag from the bottom-right. (only when not in fullscreen)
    • To toggle fullscreen, press the yellow button.
    • To move this window, drag the blue bar at the top. (only when not in fullscreen)
    • To close this window, press the red button.
